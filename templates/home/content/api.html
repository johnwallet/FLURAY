api/v-1/post/deposit/order/create/

Создание заявки на пополнение

Статус ответа

201 Created

Headers

"Key-Merchant": {Ключ мерчанта}

Body

{ "order_pay":{ID Платежной системы}, "order_amount":{Сумма}, "order_criterion":{ID Критерия} }

{ "number": "9430445", "data": "2021-03-18T20:22:05.598242", "type": "Заявка на пополнение", "pay_id": 42, "pay_name": "BITCOIN", "valute": "BTC", "criterion": "БЫСТРАЯ ЗАЯВКА", "status": "Ожидает оплаты", "requisites": "-", "amount": "0.01560000", "commission": "5.00", "exchanger": "obmen2", "total_amount": "733.72338000", "curse": "49509.0000" }

POST

api/v-1/post/deposit/order/update/

Обновление статуса у заявки на пополнение

Статус ответа

202 Accepted

Headers

"Key-Merchant": {Ключ мерчанта}

Body

{ "order_number":{Номер заявки на пополнение} }

Пример ответа

{ "number": "4624986", "status": "Оплачена" }

POST

api/v-1/post/all/order/check/

Получение статуса заявки

Статус ответа

200 OK

Headers

"Key-Merchant": {Ключ мерчанта}

Body

{ "order_number":{Номер заявки на пополнение} }

Пример ответа

{ "number": "9466837", "status": "Оплачена" }

Ошибки

Статус ответа

403 Forbidden

Body ответа

{ "ERROR": "Key 'Key-Merchant', not found" }

Решение

В Headers запроса не передан секретный ключ мерчанта. Или допущена ошибка в названии ключа, верно - Key-Merchant.

Статус ответа

400 Bad Request

Body ответа

{ "ERROR": "Secret key not found" }

Решение

Секретный ключ мерчанта не верный.

Статус ответа

400 Bad Request

Body ответа

{ "ERROR": "The data is not valid" }

Решение

Данные не валидны.

Статус ответа

400 Bad Request

Body ответа

{ "ERROR": "The specified criterion does not exist" }

Решение

Критерий заявки не найден. Принимаемые ID критерии 1, 2.

Статус ответа

400 Bad Request

Body ответа

{ "ERROR": "The specified payment system does not exist, does not exist" }

Решение

Платежная система не найдена, Принимаемые ID Платежных систем 11-49.

Статус ответа

400 Bad Request

Body ответа

{ "ERROR": "Requested amount, exceeds the minimum limit" }

Решение

Превышена минимальная сумма для заявки. Рекомендуем передавать уже валидированную сумму.

400 Bad Request

Body ответа

Решение

Превышена максимальная сумма для заявки. Рекомендуем передавать уже валидированную сумму.

Статус ответа

400 Bad Request

Body ответа

Если не нашли ответа на свой вопрос, или нужна помощь, обратитесь в Техническую поддержку.
